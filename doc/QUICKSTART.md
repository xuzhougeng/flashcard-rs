# JP Desktop 快速启动指南

## 📦 项目已完成

桌面版日文学习应用已经构建完成！

## 🚀 如何启动

### 方式1：直接运行可执行文件
```bash
.\target\release\jp-desktop.exe
```

### 方式2：使用 Cargo 运行
```bash
cargo run --bin jp-desktop --features desktop --release
```

## ✨ 功能说明

### 已实现功能
- ✅ **自动定时弹窗**：每隔设定的时间自动弹出学习卡片
- ✅ **混合学习模式**：随机显示假名或中文词汇翻译
- ✅ **精美UI**：渐变紫色设计，与Web版一致
- ✅ **手动刷新**：点击按钮可随时切换到下一张卡片
- ✅ **设置面板**：
  - 提醒间隔（5/10/15/20/30/60分钟可选）
  - 开机自启动（开/关）
  - 卡片类型（混合/仅假名/仅中文词）

### 当前限制
- ⏳ 系统托盘功能暂未实现（Tauri 2.0 API变更）
- ⏳ 设置暂未持久化保存（重启后恢复默认值）

## 🎮 使用方法

1. **启动应用**
   - 双击 `jp-desktop.exe` 或使用命令行启动
   - 应用会立即显示一张学习卡片

2. **查看学习卡**
   - 假名卡片：显示罗马字、平假名、片假名和例词
   - 中文词卡片：显示中文和对应的日文翻译及读音

3. **手动切换**
   - 点击「🔄 下一张卡片」按钮查看新卡片

4. **调整设置**
   - 点击「⚙️ 设置」按钮
   - 选择提醒间隔（默认10分钟）
   - 选择卡片类型（默认混合）
   - 设置开机自启（默认关闭）
   - 点击「保存设置」

5. **定时提醒**
   - 应用会在后台运行
   - 每隔设定时间自动弹出窗口显示新卡片
   - 关闭窗口不会退出程序，只是隐藏

## 🐛 问题排查

### 问题：窗口不显示
- 检查是否有杀毒软件拦截
- 尝试以管理员身份运行
- 查看任务管理器中是否有 jp-desktop.exe 进程

### 问题：卡片不显示内容
- 打开开发者工具（F12）查看控制台错误
- 检查 desktop-ui/ 目录是否存在且包含所有文件

### 问题：设置保存后无效
- 当前版本设置未持久化，重启后会恢复默认值
- 可在代码中修改 Settings::default() 来更改默认值
