# 日文学习 Flash Card - 网页版

基于 Rust CLI 版本改编的网页版日文学习工具，提供交互式的假名学习和中日词汇翻译功能。

## ✨ 功能特性

### 1. 假名查询（Romaji Lookup）
- 输入罗马字（如：chi, tsu, a, ka）查看对应的平假名和片假名
- 显示大字体的假名字符，便于学习和记忆
- 每个假名提供 3 个例词，包含汉字、假名读音和英文释义

### 2. 中日翻译（Chinese to Japanese）
- 输入中文词汇，查看日语翻译
- 包含假名注音和罗马字读音
- 内置 60+ 常用词汇，包括：
  - 问候语：你好、谢谢、对不起、再见等
  - 数字：一到十
  - 家庭成员：父亲、母亲、兄弟姐妹等
  - 颜色：红、蓝、白、黑、黄、绿
  - 四季：春夏秋冬
  - 星期：周一到周日
  - 常用词汇：学习、朋友、家、学校等

### 3. 智能搜索提示
- 输入时自动显示匹配的建议
- 支持罗马字和中文词汇的模糊匹配
- 点击建议快速查询

## 🚀 使用方法

### 本地运行

1. **直接打开**
   - 双击 `index.html` 文件即可在浏览器中打开
   - 或右键选择用浏览器打开

2. **使用本地服务器（推荐）**
   ```bash
   # 使用 Python 3
   python -m http.server 8000

   # 使用 Node.js (需要先安装 http-server)
   npx http-server

   # 使用 PHP
   php -S localhost:8000
   ```
   然后访问 `http://localhost:8000`

### 查询示例

1. **查询假名**
   - 输入：`chi` → 显示 ち (平假名) 和 チ (片假名)
   - 输入：`tsu` → 显示 つ (平假名) 和 ツ (片假名)
   - 输入：`a` → 显示 あ (平假名) 和 ア (片假名)

2. **查询翻译**
   - 输入：`你好` → 显示 こんにちは (konnichiwa)
   - 输入：`谢谢` → 显示 ありがとう (arigatou)
   - 输入：`春天` → 显示 春 (はる/haru)

## 📝 与 Rust CLI 版本的对应关系

| 功能 | CLI 版本 | Web 版本 |
|------|---------|---------|
| 假名查询 | `jp chi` | 在搜索框输入 `chi` |
| 中日翻译 | `jp 你好` | 在搜索框输入 `你好` |
| ASCII 艺术字 | 终端显示 ASCII 艺术 | 网页显示大字体假名 |
| 词典数据 | Rust HashMap | JavaScript 对象 |
| 搜索建议 | 无 | 实时搜索建议 |
| LLM 翻译 | 支持（需配置 API） | 待实现 |
